- No operator-facing command or subcommand requires `FOREMAN_BACKEND=node` for normal product behavior.
- Existing guarded legacy paths are either implemented through Elixir-backed APIs/events/projections or removed with clear operator-facing messaging/docs.

- Node CLI/frontend and Elixir-launched Node/Pi worker bridge remain functional where they are still required.

- In scope: deleting obsolete Node backend/server/orchestrator/store code when no longer needed by the default Elixir-backed product.
- In scope: adding Elixir APIs/events/projections/CLI plumbing needed to close those paths.
